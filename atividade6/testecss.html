Propriedades do CSS

A linguagem utilizada atualmente para definir o layout dos documentos HTML é a CSS, uma abreviação para o termo em inglês Cascading Style Sheet que, traduzido para o português, ficou como folhas de estilo em cascata.

Nesse site iremos observar em prática o uso de cinco propriedades no CSS da tag "tr" utilizada na tag "table". Uma propriedade CSS é uma característica (como a cor) cujo valor define o aspecto de como o navegador deve exibir o elemento.

Propriedade 1

A propriedade text-align é utilizada na formatação de textos, sendo utilizada para determinar a posição da margem.
Nessa célula foi usada a propriedade "text-align:center" para centralizar a margem do texto.

Propriedade 2

A propriedade border-color é utilizada em geral para alterar a coloração da borda da célula.
Nessa célula foi usada a propriedade "border-color:blue" para mudar a cor da borda para azul.

Propriedade 3

A propriedade background-color é utilizada em geral para mudar a cor de fundo de células, páginas.
Nessa célula foi usada a propriedade "background-color:gray" para mudar a cor do fundo do conteúdo da célula para cinza.

Propriedade 4

A propriedade colspan é utilizada para unir células/colunas em uma mesma linha
Nessa célula foi usada a propriedade "colspan" para unir duas células da mesma linha.

A propriedade rowspan é utilizada para unir células/colunas em linhas abaixo.
